#  (Translation Glossary)



This document provides standard translation references for key terminology in the project to ensure consistency and accuracy for translation contributors.

##  (Core Concepts)

|  | English |  | Description |
|------|---------|------|-------------|
|  | Ratio |  | Multiplier factor used for price calculation |
|  | Token | API | API access credentials or text units processed by models |
|  | Channel | API | Access channel for API service providers |
|  | Group |  | Classification of users or tokens, affecting price ratios |
|  | Quota |  | Available service quota for users |

##  (Model Related)

|  | English |  | Description |
|------|---------|------|-------------|
|  | Prompt |  | Model input content |
|  | Completion |  | Model output content |
|  | Input/Prompt |  | Content sent to the model |
|  | Output/Completion |  | Content returned by the model |
|  | Model Ratio |  | Billing ratio for different models |
|  | Completion Ratio |  | Additional billing ratio for output content |
|  | Price per call |  | Fixed price per call |
|  | Pay-as-you-go |  | Billing based on usage |
|  | Pay-per-view |  | Fixed price per invocation |

##  (User Management)

|  | English |  | Description |
|------|---------|------|-------------|
|  | Root User |  | Administrator with highest privileges |
|  | Admin User |  | System administrator |
|  | Normal User |  | Regular user with standard privileges |

##  (Recharge & Redemption)

|  | English |  | Description |
|------|---------|------|-------------|
|  | Top Up |  | Add quota to account |
|  | Redemption Code |  | Code that can be redeemed for quota |

##  (Channel Management)

|  | English |  | Description |

|  | Weight |  | Load balancing weight |

##  (Security Related)

##  (Billing Related)

|  | English |  | Description |
|------|---------|------|-------------|
|  | Ratio |  | Multiplier factor used for price calculation |

##  (Translation Guidelines)

- ** (Prompt)** =  / Model input content
- ** (Completion)** =  / Model output content
- ** (Ratio)** =  / Multiplier factor for price calculation
- ** (Quota)** =  Credit / Available service quota for users, sometimes also translated as Credit
- **Token** =  / Depending on context, may refer to:
  - API (API Token)
  -  (Text Token)
  -  (Access Token)
